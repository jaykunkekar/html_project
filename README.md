# MultiPage Website

## Overview
This project is a multipage website built with HTML, CSS, and Bootstrap.
It includes a polished, responsive design with separate pages for Home, About, Services, Gallery, and Contact.

## Files
- `index.html` — Home page with hero, carousel, and feature highlights
- `about.html` — About page with team section and company values
- `services.html` — Services page with service cards and pricing plans
- `gallery.html` — Gallery page with a filterable project grid
- `contact.html` — Contact page with a form and location map
- `styles.css` — Shared stylesheet for consistent styling

## Usage
1. Open the `MultiPage_Website` folder.
2. Open `index.html` in your web browser.
3. Use the navigation menu to move between pages.

## Notes
- All pages are linked together using relative paths.
- The website uses Bootstrap 5.3 via CDN for layout and responsive behavior.
- Update the placeholder links and images as needed for your content.

## Customization
- Change the brand name by editing the navbar text in each HTML file.
- Replace the sample images with your own by updating the `src` attributes.
- Modify `styles.css` to adjust colors, typography, and spacing.
