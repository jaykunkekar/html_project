# Core Concepts Used in This Project

This project demonstrates several fundamental web development concepts, including HTML structure, CSS styling, responsive design, and basic Git version control.

## HTML Concepts
- Document structure using `<!DOCTYPE html>`, `<html>`, `<head>`, and `<body>`
- Meta tags for responsive design: `<meta charset="UTF-8">` and `<meta name="viewport" content="width=device-width, initial-scale=1.0">`
- Semantic sections: `<nav>`, `<section>`, `<footer>`
- Navigation menu with anchor links between pages
- Use of image elements: `<img>` with `alt` text
- Form structure in `contact.html`: `<form>`, `<input>`, `<textarea>`, `<button>`, `<select>`
- Relative linking between multipage files: `index.html`, `about.html`, `services.html`, `gallery.html`, `contact.html`

## CSS Concepts
- External stylesheet usage via `<link rel="stylesheet" href="styles.css">`
- CSS variables with `:root` for consistent theming
- Custom typography, spacing, and color styling
- Responsive design via media queries (`@media`) for smaller screens
- Card styling with shadows, rounded corners, and hover transitions
- Background gradients and overlays for a modern visual style
- Button styling and hover effects
- Consistent footer and social link styling

## Bootstrap Concepts
- Bootstrap CSS CDN integration for quick styling
- Responsive grid layout with `.container`, `.row`, and `.col-*`
- Responsive navbar using `.navbar`, `.navbar-expand-lg`, `.navbar-toggler`, and collapse behavior
- Button classes like `.btn`, `.btn-primary`, `.btn-light`, and utility spacing classes
- Carousel component markup with `.carousel`, `.carousel-item`, and controls

## JavaScript Concepts
- Simple event handling for the contact form submission in `contact.html`
- DOM event listener: `addEventListener('submit', ...)`
- Form reset and success message feedback using JavaScript
- Gallery filtering logic using data attributes and button click handlers in `gallery.html`

## Responsive & UX Concepts
- Mobile-first layout adjustments using media queries
- Flexible card layout that adapts to desktop and mobile screens
- Interactive hover states to improve user engagement
- Clear page navigation and call-to-action buttons
- Visual consistency across all pages
